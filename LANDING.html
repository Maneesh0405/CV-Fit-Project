Smart Resume Screening & Matching System

About

This repository contains a Streamlit-based resume screening app (app.py). It is intended to be run as a Python Streamlit app, not served as a static site. If you opened this project on a static host (for example, Vercel static hosting) you will see a static page instead of the running app.

Quick Links

  • Run locally: instructions below
  • Deploy (recommended): use Streamlit Cloud, Render, or any Docker-capable host.

Run Locally

From the repository root run:

python -m pip install -r requirements.txt
streamlit run app.py

If you use spaCy features, install the language model:

python -m spacy download en_core_web_sm

Docker

There is a provided Dockerfile to run the app in a container. Build and run with:

docker build -t cv-fit .
docker run -p 8501:8501 cv-fit
